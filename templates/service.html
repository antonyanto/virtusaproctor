Services

BPO PROCTOR

Third party Intervention

Ensures that no one else other than the authorized employee access the data. This feature helps to prevent malpractice like face spoofing where someone else tries to pretend to be the employee. Helps in preventing the data from being accessed by an unauthorized person.

Mobile & Camera Detection

Detects the presence of restricted items like Mobile phone, camera etc. which can be used to capture the data. These devices can be misused by the employee to capture the data and send it to some unauthorized person which can compromise the data security.

USB Storage Detection

Checks if any external storage device like pen drive is connected to the device which can be used to store the copy of the data. The external storage devices can be used to to copy the data from the system and can be transferred to anyone outside the organization

Secret Snipper

This feature enables in monitoring every activity of the employee. It can monitor each and every action of the user like applications running in the employees system, screenshots taken during proctoring session. This feature also gives the admin the control over the system like disabling controls and terminating tasks if suspicious activity is detected.
