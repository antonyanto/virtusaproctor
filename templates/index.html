Eye ball Tracking

This feature ensures that the candidate isn’t deviating his focus from the screen. It helps to monitor whether the candidate indulges in any unfair means while attempting the exam like referring to any alternate resource or looking at someone to get the answers.

Object detection

This feature helps to recognize if someone else other than the candidate is present in the room. It can detect objects like mobile phones, headsets etc which are not permitted during the examination.

Mouth Tracking

This feature monitors if the candidate is speaking/ communicating with someone else during the examination.

Face spoofing.

The feature detects whether the face detected using the webcam is real or a photograph of the candidate.

Head pose estimation

This feature estimates the posture of the candidate. If the candidate tries to refer to any alternate resource during the examination the change in head pose will be detected , which indicates malpractice.

Speech to text and common word detection.

This feature records the voice of the candidate and converts it to text. The same will be compared with the keywords in the question paper. If the words spoken by the candidate match with the key words in the question paper this indicates malpractice.

Third party Intervention

Ensures that no one else other than the authorized employee access the data. This feature helps to prevent malpractice like face spoofing where someone else tries to pretend to be the employee. Helps in preventing the data from being accessed by an unauthorized person.

Mobile & Camera Detection

Detects the presence of restricted items like Mobile phone, camera etc. which can be used to capture the data. These devices can be misused by the employee to capture the data and send it to some unauthorized person which can compromise the data security.

USB Storage Detection

Checks if any external storage device like pen drive is connected to the device which can be used to store the copy of the data. The external storage devices can be used to to copy the data from the system and can be transferred to anyone outside the organization.

Secret Snipper

This feature enables in monitoring every activity of the employee. It can monitor each and every action of the user like applications running in the employees system, screenshots taken during proctoring session. This feature also gives the admin the control over the system like disabling controls and terminating tasks if suspicious activity is detected.

Speech to text and common word detection.

Recognizes the speech of the user and converts them to text. This helps to detect if the employee tries to communicate any confidential detail with someone during the session. If the words match up with the keywords, this indicated a malpractice.
